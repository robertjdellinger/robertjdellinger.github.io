## Required Image

### meet-francisca.jpg

**Description:** "Meet Francisca" - Study Abroad photo from Guatemala, Ecological and Social Issues at Lake Atitlan (UC Davis Photo Contest Winner)

**Image URL:** https://github.com/user-attachments/assets/5e810ed2-253d-47ed-b1d2-723c6ea40abb

**How to add:**

1. Download the image from the URL above
2. Save it as `meet-francisca.jpg` in this directory (`content/gallery/media/gallery/`)
3. Uncomment the image reference in `content/gallery/_index.md` (around line 44)

**More info:** https://www.ucdavis.edu/news/download-study-abroad-photo-contest
