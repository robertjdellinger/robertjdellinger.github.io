# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub community health files (SECURITY.md, CONTRIBUTING.md, CODE_OF_CONDUCT.md)
- Feature request issue template
- Pull request template
- NPM ecosystem to Dependabot configuration
- Changelog file

### Changed
- Enabled robots.txt generation in Hugo configuration
- Updated minimum Hugo version requirement to 0.152.0
- Enabled canonifyURLs for better SEO
- Improved repository organization and documentation

## [1.0.0] - 2025-11-01

### Initial Release
- Hugo-based academic website using HugoBlox template
- Personal information, research, publications, and teaching sections
- Blog and outreach content
- Gallery with marine life and research photos
- Automated deployment via GitHub Actions and Netlify
- Link checking and HTML linting workflows
- Automated publication imports from ORCID
